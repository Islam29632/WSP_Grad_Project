# How to contribute
1. Fork the repository 
```bash
git clone https://github.com/Islam29632/WSP_Grad_Project.git
```
2. install dependencies
```bash
pip install -r requirements.txt
```
3. Make your changes
4. Commit your changes
```bash
git add .
git commit -m "your message"
```
5. Push 
```bash
git push
```

# Run streamlit app
```bash
streamlit run app.py
```
